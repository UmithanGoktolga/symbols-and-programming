Axiom of Specification and Ruseell's Paradox

The Axiom of Specification states that given any set AA and any property P(x)P(x), there is a set B={x∈A|P(x)}B = \{x \in A\, |\, P(x) \}.

Notably, the axiom does not impose any restriction on the property P(x)P(x). It was Bertrand Russell, who discovered an important corollary of this lack of restriction. Take P(x)P(x) as x∉xx\not\in x. Given an arbitrary set AA, the Axiom of Specification guarantees that there is a set B={x∈A|x∉x}B = \{x\in A\, |\, x\not\in x\}. Given that BB is a well-defined set, given any object yy, it should be possible to answer whether y∈By\in B or not: for all yy, y∈By\in B iff y∈Ay\in A and y∉yy\not\in y. Now we can ask whether B∈BB\in B or not. If B∈BB\in B, then B∉BB\not\in B, therefore we arrive at a contradiction. We must conclude that B∉BB\not\in B. Knowing that B∉BB\not\in B, then we must conclude B∉AB\not\in A, since if it were, then BB would be in BB, leading again to a contradiction. Therefore, given an arbitrary set AA, there is always at least one mathematical object, the set BB constructed as above, that is not included in AA. This shows that there can be no set that includes everything, there will always be a well-defined object left out. (Halmos Ch. 2)
